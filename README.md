RWBookStore
===========

一个简单的网上书店。

Get Start
---------

获取 RWBookStore

*   下载最近的版本或者克隆库`git@github.com:RWBookstore/.git`

获取 Node.js

*   从[http://nodejs.org](http://nodejs.org)获取最新版本的Node.js

获取依赖模块

*   在应用下执行`npm install`

How to use
----------

通过`node init.js`来初始化数据库。

通过`node app.js`启动应用。
